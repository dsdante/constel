# Constel
A galaxy model

## Requirements
Arch Linux:  
\# pacman -S glew glfw-x11 freetype2

Debian/Ubuntu (presumed):  
\# apt-get install libglew-dev libglfw3-dev libfreetype6-dev

## Control
Mouse dragging: pan  
Mouse wheel: zoom  
F: fullscreen  
Physical and visual options can be set in constel.conf.
